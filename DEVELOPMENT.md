# BUILDTIME REQUIREMENTS

* [Go](https://golang.org/) 1.17+
* [accio](https://github.com/mcandre/accio) 0.0.2
* [Docker](https://www.docker.com/) 20.10.12+

# INSTALL

```console
$ mage install
```

# UNINSTALL

```console
$ mage uninstall
```

# LINT

```console
$ mage lint
```

# TEST

```console
$ mage test
```

# PORT

```console
$ mage port
```
